Model keeps asking questions until it reaches a final answer

logistic regression does only math
decision trees ask questions after questions 

real world uses-
bank loan approval
disease prediction
fraud detection
predictive maintenance

strengths-
easy to understand, can literally draw the Model
works with non linear data
little data preperation, often scaling isnt needed, normalization isnt required

weaknesses-
overfitting, biggest problem
unstable, small change can create a different tree

CORE TERMS-
root node-starting point
branch-path taken
internal node-another question
leaf node-final answer
