🔹 1. OOPS (Object-Oriented Programming)
Core Concepts
1. Class & Object
Class = blueprint
Object = instance of class
class Car:
    def __init__(self, brand, speed):
        self.brand = brand
        self.speed = speed

car1 = Car("Tesla", 200)


2. Encapsulation
Hiding internal data
Use _ or __ for private variables
class Bank:
    def __init__(self, balance):
        self.__balance = balance


3. Inheritance
Reuse code from parent class
class Animal:
    def speak(self):
        print("Sound")

class Dog(Animal):
    def bark(self):
        print("Bark")


4. Polymorphism
Same function, different behavior
def add(a, b):
    return a + b


5. Abstraction
Hide complexity, show only essentials
Use abstract classes (abc module)

🔹 3. PANDAS
What is Pandas?
Library for data analysis
Works with:
Series (1D)
DataFrame (table)

Creating DataFrame-
import pandas as pd
df = pd.DataFrame({
    "name": ["A", "B"],
    "marks": [90, 80]
})

Basic Inspection
df.head()
df.info()
df.describe()
df.columns

Selection
df["marks"]
df[["name", "marks"]]

Filtering
df[df["marks"] > 85]

Adding Columns
df["grade"] = df["marks"] * 0.1

Apply / Map
df["marks"].map(lambda x: x * 2)

df.apply(lambda row: row["marks"] + 5, axis=1)


GroupBy (CRITICAL 🔥)
df.groupby("name")["marks"].mean()

🔥 FULL PANDAS CASE STUDY (WITH JOINS)
🎯 Problem

You have 3 datasets:

Customers
Orders
Payments

👉 Goal:
Create a final cleaned dataset with:
Customer info
Order info
Payment info
Cleaned numeric values

Step 1: Create Data
import pandas as pd

df_customers = pd.DataFrame({
    "customer_id": [1, 2, 3],
    "name": ["Alice", "Bob", "Charlie"]
})

df_orders = pd.DataFrame({
    "order_id": [101, 102, 103],
    "customer_id": [1, 2, 1],
    "amount": ["100", "200", "150"]
})

df_payments = pd.DataFrame({
    "order_id": [101, 102],
    "status": ["Paid", "Pending"]
})

Step 2: Merge (JOIN DATA)

df_join1 = pd.merge(df_orders, df_customers, on="customer_id", how="left")
df_final = pd.merge(df_join1, df_payments, on="order_id", how="left")

Step 3: Clean Data
df_final["amount"] = pd.to_numeric(df_final["amount"])

Step 4: Feature Engineering
df_final["amount_with_tax"] = df_final["amount"] * 1.18

Step 5: Handle Missing Values
df_final["status"].fillna("Unknown", inplace=True)

Step 6: Analysis
Total spent per customer
df_final.groupby("name")["amount"].sum()

Average order value
df_final["amount"].mean()


Final Output Example
name	order_id	amount	status
Alice	101	100	Paid
Bob	102	200	Pending
Alice	103	150	Unknown
